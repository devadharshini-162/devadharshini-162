## stuff i've built 🚀

### 🏦 [StatementAI — Bank Statement Parser](https://github.com/devadharshini-162/bank-statement-parser)
`FastAPI · React.js · Docker · GitHub Actions`

upload your bank statement → it figures out if it's scanned or digital → parses everything → categorises transactions → flags weird stuff → exports to Excel. works for SBI, HDFC and Axis. the kind of project that started as "how hard could this be" and then became a whole thing.

---

### 🔭 [Pulsar Detection](https://github.com/devadharshini-162/Pulsar-Detection)
`TensorFlow · CNN · LSTM · Attention · GAN · SMOTE`

the dataset was 91% "not a pulsar." a bad model would just predict "not a pulsar" always and get 91% accuracy and call it a day. i didn't do that. trained a GAN to generate realistic synthetic pulsar data, stacked a CNN-LSTM-Attention model, got it actually working. research paper out of it. space is genuinely fascinating.

---

### 🚁 [Mission Planner Water Sampler Plugin](https://github.com/devadharshini-162/MissionPlanner-WaterSampler-Plugin)
`C# · .NET · ArduPilot`

real-time plugin for controlling autonomous drone water-sampling missions from a ground station. drop it in, it works, no touching the core codebase. i was very pleased with this. MIT licensed.

---

### 🏥 MediQueue — OPD Queue System
`Spring Boot · React.js · MySQL`

hospital OPD token management system. patients register, get tokens automatically, admin sees a live colour-coded dashboard that refreshes every 10 seconds. spring boot with proper layered architecture because i was in that phase.
